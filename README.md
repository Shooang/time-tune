## 🎙 产品简介

**时光调频（TimeTune）** 是一款面向家庭情感连接的 AI 年代广播产品。子女为父母/祖辈选择一个年代，老人只需旋转复古收音机上的旋钮，即可听到当年的新闻、天气预报、革命歌曲、诗词朗诵——配合 6 层真实收音机噪音与极致拟物化界面，让长辈在熟悉的声音中找回青春记忆，也让陪伴从"没话聊"变成"一起听"。

灵感来源于真实故事：很多家庭并不缺陪伴，缺的是一个能让交流重新开始、又不会增加老人使用负担的入口。

> **核心理念**：旋钮一转，时光倒流。让声音成为代际之间的情感桥梁。

## 🎨 设计亮点

- **拟物化交互**：复古收音机外观 + 物理旋钮手感（惯性、阻尼、机械联动），老人零学习成本
- **声音分层设计**：主播播报 → 背景音乐 → 环境底噪，三层动态混音营造沉浸感
- **真实广播质感**：6层物理噪音模型+速度敏感调谐算法+AM信号衰落，复刻1950年代电子管收音机听感
- **长辈零操作**：分享链接点开即听，无需注册/登录/学习操作
- **年代语言指纹**：不同年代使用不同语气、称呼、高频词（建国初期"同志们"→大跃进"鼓足干劲"）
- **适老化设计**：大字体、高对比度（WCAG标准）、大点击区（≥80×80px）

## 🏗 架构设计

Demo 版本采用**全预生成音频+实时噪音合成**架构：

```
旋钮旋转（连续月份级精度，1949.1-1960.12）
        ↓
指针位置 + 旋转速度计算
        ↓
节目播放引擎（开场问候→日期报时→新闻/天气/诗词/life加权随机→年代歌曲→循环）
        ↓
Web Audio API 实时混音：
├── 6层收音机噪音（速度敏感音量/Q值/中心频率/LFO调制）
├── 预生成语音节目（新闻/天气/诗歌/life/问候/台呼）
├── 年代背景音乐BGM（低音量铺垫，事件触发自动切换）
└── 增益包络（5ms超快淡出防重叠/1.5s淡入/停转800ms噪音衰减）
```

## 📊 学术依据

基于怀旧疗法（Reminiscence Therapy）循证研究：
- 怀旧干预可使老年人抑郁评分降低 **23-30%**（Chiang et al., 2010）
- 音乐唤回自传体记忆的成功率达 **65%**（Baird & Samson, 2014）
- 熟悉音乐对阿尔茨海默症患者的自主神经激活有显著效果（Garrido et al., 2018）
