## 🎙 产品简介

**时光调频（TimeTune）** 是一台双频道 AI 复古收音机：

- **公共频道**（已实现）：老人旋转旋钮就能听到自己年轻时代的集体记忆广播——新闻、天气、物价、流行歌，年代语言原汁原味。
- **家庭频道**（规划中）：让家人录制和保存彼此的原音，形成跨代传承的声音遗产。

灵感来自四个真实故事：93 岁的外婆一个人在家想听熟悉的声音却找不到；一位姑奶过年时笑着说"可能活不到明年了"，随手录下的视频第二年就成了她最后的影像；93 岁阿尔茨海默老兵记忆停留在 19 岁；人们在电商平台上买自己出生那天的报纸。声音是记忆的开关，也是不可再生的家族记忆。

旋钮一转，时光倒流——让长辈在熟悉的声音中不再孤独，也让家人的声音有处可留。

## 🚀 本地运行

### 环境要求

- Node.js 18+
- npm 或 pnpm

### 步骤

```bash
# 1. 克隆仓库
git clone https://github.com/Shooang/time-tune.git
cd time-tune

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 构建生产版本
npm run build
```

构建产物在 `dist/` 目录，可直接部署到任何静态托管平台。

### 音频内容说明

Demo 的音频内容（新闻、天气、诗歌、年代歌曲等共 2000+ 条）由 Python 脚本预生成，体积较大（约 500MB），不包含在仓库中。`scripts/` 目录下提供了完整的音频生成脚本，使用 edge-tts（免费，无需 API Key）生成。

如需本地完整运行，可按 `scripts/` 中的脚本自行生成音频文件到 `public/audio/` 目录。

## 🏗 架构设计

Demo 版本采用**全预生成音频 + 实时噪音合成**架构：

```
旋钮旋转（连续月份级精度，1949.1-1960.12）
        ↓
指针位置 + 旋转速度计算
        ↓
节目播放引擎（开场问候 → 日期报时 → 新闻/天气/诗词/life 加权随机 → 年代歌曲 → 循环）
        ↓
Web Audio API 实时混音：
├── 6层收音机噪音（速度敏感音量/Q值/中心频率/LFO调制）
├── 预生成语音节目（新闻/天气/诗歌/life/问候/台呼）
├── 年代背景音乐BGM（低音量铺垫，事件触发自动切换）
└── 增益包络（5ms超快淡出防重叠 / 1.5s淡入 / 停转800ms噪音衰减）
```

## 🗺 产品路线图

### 公共频道（集体年代广播）—— Demo 已实现
- **内容扩充**：扩展至 1949-1985 完整年代范围，补充小说连播/戏曲选段
- **分享功能**：子女端生成分享链接，长辈点开即听（零操作）
- **体验优化**：持续打磨旋钮手感、噪音质感、节目衔接

### 家庭频道（原音存储/声音遗产）—— 规划中
- **一键录音**：家人原音录制 + 自动降噪 + AI标注日期/说话人
- **永久存储**：不依赖社交账号的私密原音保存，对标微信语音不可继承的痛点
- **家庭共享**：一个家庭账号，多位成员可录音/回放

### 长期方向
- **实体 AI 收音机**：终局形态，通过养老机构/社区落地，触达 80-90 岁高龄老人
- **小程序版本**：微信生态内更顺畅的分享和使用体验

## 📊 学术依据

基于怀旧疗法（Reminiscence Therapy）循证研究：
- 怀旧干预可使老年人抑郁评分降低 23-30%（Chiang et al., 2010）
- 音乐唤回自传体记忆的成功率达 65%（Baird & Samson, 2014）
- 熟悉音乐对阿尔茨海默症患者的自主神经激活有显著效果（Garrido et al., 2018）
