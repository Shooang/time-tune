## 🎙 产品简介

**时光调频（Time Tune）** 是一款为中老年人设计的 AI 年代广播产品。用户只需旋转复古收音机上的频率旋钮，即可穿越回 1949-1960 年代的广播时光（Demo 当前阶段，未来规划扩展至 1949-2000 年）——听到当年的新闻播报、革命歌曲、时代旋律，配合 6 层真实收音机噪音与极致拟物化界面，让老年人在熟悉的声音中找回青春记忆。

> **核心理念**：旋钮一转，时光倒流。让声音成为老年人的时光机。

## ✨ 已实现功能（Demo）

| 功能 | 说明 |
|------|------|
| 🎛 **复古收音机旋钮** | 拟物化频率旋钮，连续调频 1949-1960 年（支持月份级精度，含橡皮筋回弹/边界阻尼/档位咔哒感） |
| 📻 **真实调谐噪音** | Web Audio API 实时生成 6 层老式收音机噪音（粉红噪+布朗噪+高频嘶嘶+50Hz嗡声+静电爆裂+AM信号衰落），速度敏感调谐算法 |
| 🔊 **年代主播播报** | 精确匹配年月的新闻广播，共 144 条月度新闻音频 + 年代节目编排 |
| 🎵 **年代歌曲片段** | 建国初期经典革命歌曲与时代旋律（歌唱祖国、东方红、志愿军战歌 等） |
| 🎼 **过场音乐衔接** | 年代背景纯音乐 BGM，语音节目时低音量铺垫，切换类型不重复 |
| 💡 **点阵年份显示** | 5×7 LED 点阵字体显示年份，暖琥珀色真空管辉光效果，开机暖机动画 |
| 🔔 **首次引导仪式** | 旋钮摇摆脉冲提示 → 触碰启动 → 电源冲击噪声 → 屏幕频闪 → 信号锁定 |
| 📱 **移动端适配** | 支持触摸操作，后台持续播放（陪伴模式），iOS 音频播放兼容 |

## 🗺 产品路线图（规划中）

Demo 仅展示核心交互体验，完整产品规划如下：

### 🎯 短期
- **❤️ 收藏频道**：本地收藏喜欢的年代
- **🎵 内容扩充**：逐步补充年代歌曲、生活片段、台呼间奏等素材，丰富节目多样性
- **🔧 体验优化**：持续打磨旋钮手感、噪音质感、节目衔接等细节体验

### 💰 中期
- **👨‍👩‍👧 子女代创建**：子女为父母创建账号，填写家乡、青春故事等信息，生成个性化内容
- **📤 分享功能**：生成专属时光卡片分享到微信
- **💎 订阅模式**：会员解锁更多年代、无广告
- **👪 家庭共享**：一个家庭账号支持多位老人使用
- **📊 倾听报告**：子女端查看父母常听年代、收听时长，了解父母精神状态
- **🏥 疗愈模式**：基于怀旧疗法（Reminiscence Therapy）研究，针对认知障碍老人的专业内容

### 🌐 长期
- **🎂 时光胶囊**：用户选择生日/纪念日具体日期，AI生成当天专属年代回忆广播（需内容库完善至日粒度）
- **🗣 实时 TTS 合成**：接入云端 TTS（火山引擎/CosyVoice 2），支持无限内容生成
- **🤖 AI 个性化主播**：学习用户偏好，动态调整节目单
- **🏘 社区功能**：同年代用户的声音回忆分享社区
- **⌚ 智能硬件**：实体复古收音机硬件，内置 WiFi 与语音控制

## 🎨 设计亮点

- **拟物化交互**：复古木质收音机外观 + 物理旋钮手感（惯性、阻尼、机械联动），降低老年人学习成本
- **声音分层设计**：主播播报 → 背景音乐 → 环境底噪，三层声音动态混音营造沉浸感
- **真实广播质感**：6层物理噪音模型+速度敏感调谐算法+信号锁定衰减，复刻1950年代电子管收音机听感
- **认知无障碍**：大字体、高对比度、操作按钮明显，符合WCAG适老化标准

## 🏗 架构设计

Demo 版本采用**全预生成音频+实时噪音合成**架构：

```
旋钮旋转（连续月份级精度，1949.1-1960.12）
        ↓
指针位置 + 旋转速度计算
        ↓
节目播放引擎（开场→当月新闻→同年新闻+life片段→年代歌曲→循环）
        ↓
Web Audio API 实时混音：
├── 6层收音机噪音（速度敏感音量/Q值/中心频率/LFO调制）
├── 预生成语音节目（新闻/life片段）
├── 年代背景音乐BGM（低音量铺垫，类型切换才换BGM）
└── 增益包络（0.15s淡出/0.3s淡入/停转800ms噪音衰减）
```

**范围扩展规划**：Demo阶段聚焦1949-1960年（建国初期），验证核心交互体验；后续迭代扩展至1949-2000年完整年代范围。

**生产环境三层架构**（规划中）：

| 层级 | 内容类型 | 响应速度 | 实现方式 |
|------|---------|---------|---------|
| 🔥 热数据 | Demo当前1949-1960预设内容 | 即时 | CDN 预生成缓存 |
| 🌤 温数据 | 用户常听年代定制 | 1-3s | Edge Function 边缘缓存 |
| ❄️ 冷数据 | 时光胶囊/个性化/扩展年代 | 10-30s | 云端 TTS + AI 广播稿实时合成 |

## 📊 数据验证

基于怀旧疗法（Reminiscence Therapy）循证研究：
- 怀旧干预可使老年人抑郁评分降低 **23-30%**（Chiang et al., 2010）
- 音乐唤回自传体记忆的成功率达 **65%**（Baird & Samson, 2014）
- 熟悉音乐对阿尔茨海默症患者的自主神经激活有显著效果（Garrido et al., 2018）
