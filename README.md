## 🎙 产品简介

**时光调频（TimeTune）** 是一台双频道 AI 复古收音机：**公共频道**让老人旋转旋钮就能听到自己年轻时代的集体记忆广播；**家庭频道**（规划中）让家人录制和保存彼此的原音，形成跨代传承的声音遗产。

灵感来自两个真实故事：93 岁的外婆一个人在家，想听熟悉的声音却找不到；一位姑奶在过年时笑着说"可能活不到明年了"，当时随手录下的视频，第二年就成了她最后的影像。声音是记忆的开关，也是不可再生的家族记忆。

旋钮一转，时光倒流——让长辈在熟悉的声音中不再孤独，也让家人的声音有处可留。

> **核心理念**：旋钮一转，时光倒流。让声音成为代际之间的情感桥梁。

## 🗺 产品路线图（规划中）

Demo 展示公共频道核心体验，完整产品采用双频道架构：

### 📻 公共频道（集体年代广播）—— Demo 已实现
- **🎵 内容扩充**：扩展至 1949-1985 完整年代范围，补充小说连播/戏曲选段
- **📤 分享功能**：子女端生成分享链接，长辈点开即听（零操作）
- **🔧 体验优化**：持续打磨旋钮手感、噪音质感、节目衔接

### 🏠 家庭频道（原音存储/声音遗产）—— 中期规划
- **🎙 一键录音**：家人原音录制+自动降噪+AI标注日期/说话人
- **💾 永久存储**：不依赖社交账号的私密原音保存，对标微信语音不可继承的痛点
- **📤 短视频导出**：支持片段导出用于抖音/视频号分享
- **👨‍👩‍👧 家庭共享**：一个家庭账号，多位成员可录音/回放

### 🌐 长期方向
- **⌚ 实体 AI 收音机**：终局形态，通过养老机构/社区落地，真正触达 80-90 岁高龄老人
- **📱 小程序版本**：微信生态内更顺畅的分享和使用体验

## 🎨 设计亮点

- **拟物化交互**：复古收音机外观 + 物理旋钮手感（惯性、阻尼、机械联动），老人零学习成本
- **声音分层设计**：主播播报 → 背景音乐 → 环境底噪，三层动态混音营造沉浸感
- **真实广播质感**：6层物理噪音模型+速度敏感调谐算法+AM信号衰落，复刻1950年代电子管收音机听感
- **长辈零操作**：分享链接点开即听，无需注册/登录/学习操作
- **年代语言指纹**：不同年代使用不同语气、称呼、高频词（建国初期"同志们"→大跃进"鼓足干劲"）
- **适老化设计**：大字体、高对比度（WCAG标准）、大点击区（≥80×80px）
- **声音遗产理念**：公共频道用集体记忆打开话匣子，家庭频道用私密原音留住家人声音——今天的日常，就是明天最珍贵的记忆

## 🏗 架构设计

Demo 版本采用**全预生成音频+实时噪音合成**架构：

```
旋钮旋转（连续月份级精度，1949.1-1960.12）
        ↓
指针位置 + 旋转速度计算
        ↓
节目播放引擎（开场问候→日期报时→新闻/天气/诗词/life加权随机→年代歌曲→循环）
        ↓
Web Audio API 实时混音：
├── 6层收音机噪音（速度敏感音量/Q值/中心频率/LFO调制）
├── 预生成语音节目（新闻/天气/诗歌/life/问候/台呼）
├── 年代背景音乐BGM（低音量铺垫，事件触发自动切换）
└── 增益包络（5ms超快淡出防重叠/1.5s淡入/停转800ms噪音衰减）
```

## 📊 学术依据

基于怀旧疗法（Reminiscence Therapy）循证研究：
- 怀旧干预可使老年人抑郁评分降低 **23-30%**（Chiang et al., 2010）
- 音乐唤回自传体记忆的成功率达 **65%**（Baird & Samson, 2014）
- 熟悉音乐对阿尔茨海默症患者的自主神经激活有显著效果（Garrido et al., 2018）
