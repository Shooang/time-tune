## 🎙 产品简介

**时光调频（Time Tune）** 是一款为中老年人设计的 AI 年代广播产品。用户只需旋转复古收音机上的频率旋钮，即可穿越到 1950-2000 年代的广播时光——听到当年的新闻播报、革命歌曲、改革开放报道、港台流行音乐等，配合白噪音与拟物化界面，让老年人在熟悉的声音中找回青春记忆。

> **核心理念**：旋钮一转，时光倒流。让声音成为老年人的时光机。

## 🗺 产品路线图（规划中）

Demo 仅展示核心交互体验，完整产品规划如下：

### 🎯 短期（1-3 个月）
- **🎂 时光胶囊**：用户输入生日/纪念日，AI 生成专属年代回忆广播
- **👨‍👩‍👧 子女代创建**：子女为父母创建账号，填写家乡、青春故事等信息，生成个性化内容
- **❤️ 收藏频道**：用户可收藏喜欢的年代和节目
- **📤 分享功能**：生成专属时光卡片分享到微信

### 💰 中期（3-6 个月）
- **💎 订阅模式**：¥19.9/月 会员，解锁更多年代、自定义内容、无广告
- **👪 家庭共享**：一个家庭账号支持多位老人使用
- **📊 倾听报告**：子女端查看父母常听年代、收听时长，了解父母精神状态
- **🏥 疗愈模式**：基于怀旧疗法（Reminiscence Therapy）研究，针对认知障碍老人的专业内容

### 🌐 长期（6-12 个月）
- **🗣 实时 TTS 合成**：接入云端 TTS（火山引擎/CosyVoice 2），支持无限内容生成
- **🤖 AI 个性化主播**：学习用户偏好，动态调整节目单
- **🏘 社区功能**：同年代用户的声音回忆分享社区
- **⌚ 智能硬件**：实体复古收音机硬件，内置 WiFi 与语音控制

## 🎨 设计亮点

- **拟物化交互**：复古收音机外观 + 物理旋钮手感，降低老年人学习成本
- **声音分层设计**：主播播报 → 过场音乐 → 歌曲 → 环境噪音，四层声音叠加营造沉浸感
- **年代声音风格**：每个年代的主播声音风格不同（50s 庄重激昂 → 80s 平实自然 → 00s 活力欢快）
- **认知无障碍**：大字体、高对比度、操作按钮明显，符合老年人使用习惯

## 🏗 架构设计

Demo 版本采用**全预生成**架构，所有音频内容提前合成并托管：

```
用户旋转旋钮 → React 状态更新 → 播放对应年代预生成音频
                          ↓
              Web Audio API 实时叠加：
              ├── 粉红噪音（调台干扰声）
              ├── 高低通滤波（根据旋钮位置动态变化）
              └── 增益包络（过场淡入淡出）
```

**生产环境三层架构**（规划中）：

| 层级 | 内容类型 | 响应速度 | 实现方式 |
|------|---------|---------|---------|
| 🔥 热数据 | 高频年代默认频道 | 即时 | CDN 预生成缓存 |
| 🌤 温数据 | 用户常听年代定制 | 1-3s | Edge Function 边缘缓存 |
| ❄️ 冷数据 | 时光胶囊/个性化 | 10-30s | 云端 TTS 实时合成 |

## 📊 数据验证

基于怀旧疗法（Reminiscence Therapy）循证研究：
- 怀旧干预可使老年人抑郁评分降低 **23-30%**（Chiang et al., 2010）
- 音乐唤回自传体记忆的成功率达 **65%**（Baird & Samson, 2014）
- 熟悉音乐对阿尔茨海默症患者的自主神经激活有显著效果（Garrido et al., 2018）
